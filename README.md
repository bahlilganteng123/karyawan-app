# 💼 Aplikasi Karyawan Laravel

Aplikasi sederhana untuk mengelola data karyawan menggunakan Laravel.

## ✨ Fitur
- CRUD Karyawan
- Relasi dengan Jabatan
- Join data (Eloquent ORM)
- Fitur Search
- UI modern dengan Tailwind CSS

## 🛠️ Teknologi
- Laravel
- MySQL
- Tailwind CSS

## 📸 Screenshot


## 🚀 Cara Menjalankan
1. Clone repo
2. composer install
3. php artisan migrate
4. php artisan serve
